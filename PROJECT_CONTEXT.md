# Task Graph 项目上下文

## 项目定位

Task Graph 是一个独立的 Obsidian 插件，用项目依赖图展示 inline Tasks，重点回答“下一步能做什么”和“任务被什么阻塞”。

## 当前实现

- 原生解析 Markdown checkbox task，不依赖 Dataview。
- 识别 Obsidian Tasks 的 `🆔`、`⛔`、状态、日期、优先级和标签。
- 主编辑区图形视图，支持项目文件切换、搜索、状态筛选、平移缩放、自动适应和节点详情。
- 前置任务指向依赖任务；自动计算可开始、进行中、被阻塞和已完成。
- 支持从图中打开原文与切换任务完成状态。
- 使用原生 HTML 节点、SVG 边和内置分层布局，不依赖 React。
- 项目范围变化时自动重新适应画布，避免沿用上一个项目的缩放比例。

## 当前边界

- MVP 只支持 inline task。
- 正式依赖只来自 `🆔` / `⛔`。
- 暂不支持拖线修改依赖、图内创建任务和嵌入图。
- 修改后至少运行 `npm run build`。
